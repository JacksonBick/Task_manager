Define CRUD.- CRUD is an acronym that stands for (Create, Read, Update, and Delete) Create - to be able to insert new info into the database, Read - to be able to get and have the info from the database, Update - to be able to change existing objects in the database, Delete - to be able to remove objects from the database

Define MVC. - Stands for (Model, View, and Controller). Model is the classes you make to be able to handle and use the database. View is the user side and what the application looks like. Controller - is the in between the view and the model. Basically model = backend, view = frontend, controller = communication between the two

What are params? Where do they come from? params are parameters that we instate for out application so that when a user sends a request to the server the controller knows how to process it

What is the purpose of a serializer? A serializer is a toll that takes the user input and converts it into a language that the server can understand
